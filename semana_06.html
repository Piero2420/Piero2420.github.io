DEFINICIONES


REACT + VITE: Vite es una herramienta de tooling para el Frontend. Lo puedes agrupar en la categoría donde se encuentran otros como Webpack, Parcel y Snowpack.

Lo que necesitamos es tener el proyecto creado:
  • Descarga NODE JS
  • Crea un proyecto con el nombre de Proy02
  • Ejecuta con el comando: npm run dev

Algunos conceptos sobre la estructura de los archivo de VITE :
  • carpeta node-modules: se pondrán las librerías
  • src asets main : tiene los componentes

Componente: Una unidad de interfaz , permite ofrecer funcionalidades, cada componente puede tener muchas funcionalidades
El DNS traduce nombre de dominio a direciones IP

  • Snippet: fragmento de codigo
  • Web componentes: son secciones de la pantalla

COMPONENTES

es una pieza autónoma y reutilizable de la interfaz de usuario. En React, los componentes pueden ser clases o funciones que retornan elementos de React. Beneficia de las ventajas de rendimiento y eficiencia de Vite durante el desarrollo.

PROOPS

"props" son la forma principal de pasar datos de un componente padre a un componente hijo en React. "Props" es una abreviatura de "propiedades". Son argumentos que se pueden pasar a los componentes de React cuando se crean y se utilizan para personalizar y configurar el comportamiento y la apariencia de esos componentes.
  • Componente padre
  • Componente hijo
