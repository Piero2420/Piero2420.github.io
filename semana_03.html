En esta semana 3 hablaremos acerca de las libreria de CSS un resumen de todo lo visto

DEFINICIONES


Framwork de CSS: Es una biblioteca de estilos de genéricos que puede ser utilizada para implementar diseños web.

Ventajas:
  • Facil de implementar diseños web
  • Funciona en varios navegadores
  • Cumple estandar y funcionabilidad

Desventajas:
  • Codigo innecesario provoca ancho de banda
  • Limitación en posibilidades de diseño

Diseño Responsivo: Llamado tambien diseño web adaptable, es un enfoque de diseño de pagina web destinado redimensionar y reubicar todos los elemento de uan pagina web de forma que su visualizació sea la adecuada.
Elementos clave de diseño Responsivo:
  • Tamaño de navegación
  • Formato de visualización
  • Contenidos en texto
  • Contenidos visuales
  • Marca o logo
  • Diseño visual

BOOTSTRAP

Biblioteca multiplataforma o conjunto de herrammientas de codigo abierto para diseño de sitios y aplicaciones web. Contiene plantilla de diseño con tipografia. Componentes:
  • Alerts
  • Buttons
  • Spinner
  • Modal
  • Nav, etc

TAILWIND

Es un framework de bajo nivel de CSS que nos permite crear diseños personalizados eliminando los estilos de los componentes predefinidos

Ventajas:
  • Utilidades similares al CSS
  • Facil uso
  • Utilidades y componentes personalisables
  • Buena documentación para el aprendizaje

Desventajas:
  • El marcado a la vista parece desorganizado
  • No es facil de aprender sino se tiene conociento de CSS
  • Te vez obligado a aprender todo desde cero
  • Si buscamos agilizar el uso no es opcion pues demora mucho

PROCEDIMIENTO


Para esta semana 3 el procedimiento que se tuvo en cuenta fue el de aprendren un poco mas acerca de CSS y framework que podemos hacer uso para dar algunos estilos a nuestra pagina web , mas facil. Es por ello, que buscamos informacion y ejercicios acerca de estos temas mencionados

Algunos de ellos son:

  • Filas y columnas con Boopstrap
  • Filas y columnas con Tailwind
  • Filas y columnas con CSS
  • Aplicar codigo de Boopstrap en la realizacion de una pagina
  • Aplicar codigo de Tailwind en la realizacion de una pagina

Luego se realizo una practica en la cual nosotros debimos realizar uso de estos framework para el estilo de la pagina.


RESULTADOS


Con esta semana 3 aprendimos a diferenciar entre los frameworks que ofrece CSS y cuales a nuestra percepción nos es mas amigables para dar estilos a nuestra página. En esta semana nosotros ya contamos con los siguientes conocimientos.

  • Tendremos conocimiento del funcionamiento del Framework Boopstrap
  • Sabremos aplicar grid para la distribucion de celdad en nuestra pagina
  • Diferenciaremos el uso puro de CSS y de los frameworks
  • Conocemos las bondades de Tailwind
  • Conocemos que facilidades nos ofrece cada Framework
