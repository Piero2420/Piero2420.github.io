Uso de Hooks en React + Vite

En esta semana 10 hablaremos acerca de los Hooks, un resumen de todo lo visto en clases.

DEFINICIONES


Hooks: Esta función en JavaScript te permite crear un componente de React y gestionar su estado y ciclo de vida.


UseState:Este hook permite agregar estado a componentes funcionales en React
Se utiliza para declarar una variable de estado y una función que permite actualizarla. Retorna un array donde el primer elemento es el estado actual y el segundo es una función para actualizarlo.

UseEffect:Este hook permite realizar efectos secundarios en componentes funcionales.
Se utiliza para ejecutar código después de que un componente se monta (se renderiza), se actualiza o se desmonta.

UseContext:Este hook permite acceder al contexto de React
Se utiliza para consumir datos que son proporcionados por un Context.Provider en la jerarquía de componentes de React

UseReducer:Este hook permite manejar el estado de un componente utilizando el patrón de reducer.
Es útil cuando el estado del componente tiene una lógica más compleja y puede ser mejor gestionado utilizando un reducer en lugar de múltiples llamadas a useState.

PROCEDIMIENTO


Para esta semana el procedimiento que se tuvo en cuenta fue el de aprendren un poco mas sobre los hooks que podemos utilizar en React + Vite mediante ejercicios dentro de Visual Code.

Algunos de estos procedimientos son:

  • Usar componentes en una web
  • Hacer una aplicación web con hooks personalizados
  • Realizar Cruds con uso de los hooks


Luego se realizo una practica en la cual nosotros debimos hacer uso REACT + VITE con hooks para el desarrollo de cada punto.

RESULTADOS


Con esta decima semana acerca de Hooks, tendremos en nosotros los siguientes resultados.

  • Tendremos conocimiento sobre la estructura que utiliza los hooks
  • Sabremos acerca de los distintos hooks y su funcionamiento
  • Conoceremos como realizar una pagina haciendo uso de Hooks
