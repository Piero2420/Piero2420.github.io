HTML Y CSS

En esta semana 2 hablaremos acerca de HTML Y CSS un resumen de todo lo visto en clases.

DEFINICIONES


OPEN WEB PLATFORM: Colección de tecnologías y estándares, web abierta. (HTML5, CSS, SVG, ...)

Entre las caracteristicas tenemos:
  • Evita ser controlado
  • Aporta transparencia
  • Capacidad de Integración
  • Libertad de uso

HTML: Es un lenguaje marcado, acrónimo en inglés de HyperText Markup Language, hace referencia al lenguaje de marcado utilizado en la creación de páginas web. Este estándar que sirve de referencia del software que interactúa con la elaboración de páginas web en sus diferentes versiones.
Hace uso de:
  • (< >) etiquetas
  • En el documento termina con .html
  • Es parte de SGML

DOM (Document Objet Model): Hace referencia a la estructura del documento
Algunas etiquetas son:
  • head (metadatos)
  • style (Css en linea)
  • nav (enlace de navegación)
  • link (css externos)
  • html ( raiz del doc )
  • section ( secciones del documento)

XML : Lenguaje marcado extensible, es un estandar que se utiliza para almacenar e intercambiar información. Esta compuesto por:
  • Prolog : nos da la versión y el encoding (metadatos)
  • Body: esta el mensaje que se quiere dar ( contenido de datos simples)

Arbol DOM: Interfaz para documentos HTML Y XML, arbol de elementos, permite leer y manipular el contenido, la estructura y estilos.
  • Render TREE: Poder de modificar el objeto
  • Arbol de Nodos: Objeto del DOM es convertir la estructura y contenido del doc HTML en un modelo de objeto que puede ser utilizado por varios programas.

BOM (Browser Object Model): Ventana gráfica, lo manipula el Windows, se conecta al navegador con JS. Ejemplo: windows.document() . Se maneja por la etiqueta script.

Ciclo de vida de una pagina web: Conocido como Critical Rendering Path, es la secuencia de datos que sigue el navegador para convertir HTML, CSS y JAVASCRIPT en pixeles en pantalla


CSS

Es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Uso de los estilos:
  • Podemos utilizarlo en la misma pagina en la etiqueta HEAD
  • En archivos CSS mediante la etiqueta LINK

Estilos CSS permite definir estilos reutilizables:
  • Usar y reutilizar clases : class=" "
  • Estilos no reutilizables mediante : id=" "

Selectores:
Entre ellos tenemos.
  • Selectores de tipo
  • Selector de clase
  • Selector universal
  • Selector de id
  • Combinador de hijo etc.

FLEXBOX

Estilo CSS utilizados para establecer el espacio de los elementos en fila o columnas de una forma dinamica y sencilla.
  • En linea: display: inline
  • En bloque: display: block
  • En tabla: display: table

GRID LAYOUT

Es un estandar de las hojas de Estilo en cascada que nos permite maquetar contenido ajustandose a una rejilla en dos dimensiones totalmente considerables
  • Se compone de lineas horizontales
  • Espacio delimitado entre dos lineas se llama track
