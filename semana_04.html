JAVASCRIPT

En esta semana 4 hablaremos acerca de JAVASCRIPT un resumen de todo lo visto

DEFINICIONES


JS: Es un lenguaje script dinamico, multiparadigma, basado en prototipos,es el lenguaje mas importante, no solo para la web

Nos permite crear:
  • Aplicaciones de servidor
  • Aplicaciones moviles
  • Aplicaciones de consola
  • Aplicaciones web

Motores de JavaScript:
  • Mozilla
  • V8 de Google
  • Carakan

Node JS: Es un entorno para JavaScript construido con el motor de Javascript V8
Las instrucciones de Javascript son llamadas sentencias y estan separadas por un punto y coma.

Donde podemos incluir Javascript
  • Incluir en el Documento HTML
  • Describirlo en archivo externo
  • Javascript tag

Entre las variables tenemos varias formas de declararlo:
  • var
  • let
  • const

Tipo de datos:
  • Boolean
  • String
  • Null
  • Symbol

Variable Hosting: Hacer referencia a una variable declara mas tarde, ain obtener una excepción.
Estructura de control(Condicionales):
  • Condicional IF
  • Condicional switch pag 10

Estructuras repetitivas: Sirven para programar bucles
  • Bucle for
  • Bucle while
  • El Bucle do while, etc

Funciones y Procedimientos

Una función en JavaScript es un bloque de codigo diseñado para realizar una tarea en particular cuando la invocan.
  • Funcion declaration: Se define con la palabra reservada function
  • Invocacion de funciones: nombre de la funcion + () dentro lista de argumentos
  • Funciones anonimas: Funciones que no tendran un nombre pero, deben ser almacenadas en variables
  • Funcion object: Definidas con un constructor de funciones de JS
  • Funciones flecha: Tratan de simplificar codigo

Cadena String

Es un cojunto de elementos de valores enteros de 16-BITS. Son texto del tipo String
  • Literales de cadena: Utilizando "" o ''
  • Objetos de cadena: Es un envoltorio al rededor de los datos.

Metodos string: Entre los mas comunes podemos mencionar
  • charArt : Devuelve el caracter en posicion especificada
  • split: Dvide un objeto string en un array de strings
  • Normalice: Normalizado en unicode
  • trim: Elimina los espacios en blanco del principio al final

Arrays, Number y modulos JS

Number: Es la clase del tipo primitivo number. Podemos representar numeros enteros, decimales, etc
  • Declarar number: var numero1
  • Funciones number: number.toFixed(x) redondea number con x decimales
  • Modulo math: Nos permite realizar operaciones matematicas Ejm: Math.random y Math.pi

Nosotros tambien podemos hacer uso de herencia de objetos, desestructuración de objetos
